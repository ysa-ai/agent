# ysa-agent

Binary releases for [ysa-agent](https://ysa.run) — the daemon that connects to your ysa.run dashboard and runs issues in sandboxed containers.

## Install

**curl:**
curl -fsSL https://raw.githubusercontent.com/ysa-ai/agent/main/install.sh | sh

**Homebrew:**
brew install ysa-ai/tap/ysa-agent

**npm (requires Bun):**
bun install -g @ysa-ai/agent
